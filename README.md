## Start

```bash
git clone https://github.com/nusr/excel.git
cd excel

npm install
npm run start
```

## Features

- Font
- Font size
- Font bold
- Font italic
- Font underline
- Text color
- Fill color
- Text wrapping
- Formulas
- Insert row, column
- Delete row, column
- Insert Sheet
- Delete Sheet
- Rename Sheet
- Hide Sheet
- Unhide Sheet
- Copy
- Cut
- Paste
