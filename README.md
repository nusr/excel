## Start

```bash
git clone https://github.com/nusr/excel.git
cd excel

npm i -g pnpm
pnpm i
npm run start
```

## Unit Test

```bash
npm run test
```

## E2E Test

```bash
pnpm exec playwright install
npm run e2e
```
